\chapter*{Colofón}
\justifying
La composición tipográfica de este libro se realizó utilizando gbpublisher.

Las familias tipográficas utilizadas dentro del libro son: IBM Plex, una superfamilia de tipografía abierta, diseñada y desarrollada conceptualmente por Mike Abbink en IBM con colaboración de Bold Monday y Libertinus, bifurcación de la fuente Linux Libertine, diseñada para el texto del cuerpo y la lectura extendida.\bigskip

Cómo citar este libro:

\begin{tcolorbox}[enhanced, % permite esquinas redondeadas
colframe=black,             % color del filete
boxrule=.5pt,               % grosor del filete
arc=2mm]                    % radio de redondeo
\fullcite{referencia-del-libro}
\end{tcolorbox}
